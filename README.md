# Task
用Node开发一个简单的命令行任务工具


新建一个任务：
```javascript
    task add [task]
```

修改一个任务：
```javascript
    task update [old task] [task]
```

查找一个任务:
```javascript
    task get [task]
```

### 删除一个任务：
```javascript
    task remove [task]
```

完成一个任务：
```javascript
    task done [task]
```

获取全部任务：
```javascript
    task list
```
